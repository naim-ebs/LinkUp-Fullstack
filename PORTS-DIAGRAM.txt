To change backend from port 3000 to 8080:
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Step 1: Update .env
    nano .env
    # Change: PORT=3000
    # To:     PORT=8080

Step 2: Update Apache config (production only)
    sudo nano /etc/apache2/sites-available/nlive.conf
    # Replace all instances of "localhost:3000" with "localhost:8080"

Step 3: Restart services
    pm2 restart nlive-backend
    sudo systemctl reload apache2

Step 4: Verify
    # Check backend is listening on new port
    sudo netstat -tlnp | grep 8080
    
    # Test connection
    curl http://localhost:8080
